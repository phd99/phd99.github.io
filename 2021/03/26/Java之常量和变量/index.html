Java之常量和变量

发表于2021-03-26|更新于2021-04-03|技术Java
|字数总计:341|阅读时长:1分钟|阅读量:

一、常量

1.1. 概述

  • 常量就是在程序运行过程中，其值不可以发生改变的量。

1.2. 常量分类

  • 字符串常量
    • 用双引号括起来的内容
    • 范例："HelloWorld"
  • 整数常量
    • 不带小数的数字
    • 范例：666，-88
  • 小数常量
    • 带小数的数字
    • 范例：13.14，-5.21
  • 字符常量
    • 用单引号括起来的内容
    • 范例：'A'，'0'，'我'
  • 布尔常量
    • 布尔值，表示真假
    • 只有两个值：true，false
  • 空常量
    • 一个特殊的值，空值
    • 值是：null
    • 空常量是不能直接输出的。

二、变量

2.1. 概述

  • 变量就是在程序运行过程中，其值可以发生改变的量。
  • 从本质上讲，变量是内存中的一小块区域。

2.2. 定义

  • 格式
    • 数据类型 变量名 = 变量值;
  • 范例
    • int a = 10;

2.3. 使用

  • 取值和修改值

2.4. 注意事项

  • (1)不能重复定义同一个变量名
  • (2)定义long类型的变量时，赋值时需要加上L
    • 例如
      • long a = 10000L
    • 因为给出的一个整数默认是int类型的
  • (3)定义float类型的变量时，赋值时需要加上f
    • 例如
      • float a = 3.14F
    • 因为给出的一个小数默认是double类型的
  • (4)变量未赋值时，不能使用
