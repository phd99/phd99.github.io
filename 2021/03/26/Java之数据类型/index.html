Java之数据类型

发表于2021-03-26|更新于2021-04-03|技术Java
|字数总计:309|阅读时长:1分钟|阅读量:

一、简介

  • java语言是强类型语言，对于每一种数据都给出了明确的数据类型，不同的数据类型也分配了不同的内存空间，所以它们表示的数据大小也是不一样的。
  • 数据类型
    • 基本数据类型
      • 数值型
        • 整数byte,short,int,long
        • 浮点数float,double
        • 字符char
      • 非数值型
        • 布尔boolean
    • 引用数据类型
      • 类class
      • 接口interface
      • 数值[]

二、数据类型内存占用和取值范围

2.1. 整数

  • byte
    • 内存占用：1
    • 取值范围：-128~127
  • short
    • 内存占用：2
    • 取值范围：-32768~32767
  • int(默认)
    • 内存占用：4
    • 取值范围：(-2的31次方~2的31次方)-1
  • long
    • 内存占用：8
    • 取值范围：(-2的63次方~2的63次方)-1

2.2. 浮点数

  • float
    • 内存占用：4
    • 取值范围
      • 负数：-3.402823E+38~-1.401298E-45
      • 正数：1.401298E-45~3.402823E+38
  • double(默认)
    • 内存占用：8
    • 取值范围
      • 负数：-1.797693E+308~-4.9000000E-324
      • 正数：4.9000000E-324~1.797693E+308

2.3. 字符

  • char
    • 内存占用：2
    • 取值范围：0~65535

2.4. 布尔

  • boolean
    • 内存占用：1
    • 取值范围：true,false

2.5. E

  • E+99表示：乘以10的99次方
  • E-99表示：乘以10的-99次方
