CSS的相关属性

发表于2021-06-02|更新于2021-06-07|技术CSS
|字数总计:1.5k|阅读时长:6分钟|阅读量:

一、字体属性

1.1. 简介

  • 字体的属性
    • 风格
    • 大小
    • 粗细
    • 样式
  • 也可以写成复合属性

1.2. 字体风格

  • 用font-family定义字体风格
  • 可以同时设置多个字体，先查询前面的字体，若前面的字体不存在，则查询后面的字体，若都不存在，则使用浏览器的默认字体。
  • 字体之间用英文逗号隔开
  • 可以用英文也可以用中文
  • 范例

1.3. 字体大小

  • 用font-family定义字体大小
  • 常用单位为px，像素
  • 范例

1.4. 字体粗细

  • 用font-weight定义字体粗细
  • normal==400（正常），bold==700（加粗）
  • 范例

1.5. 字体样式

  • 用font-style定义字体样式
  • normal正常值
  • italic斜体
  • 范例

1.6. 复合属性

  • 用font定义复合属性
  • 可以将字体属性按固定的顺序写在一起
  • 按样式style、粗细weight、大小size、风格family的顺序设置，各个属性之间以空格隔开
  • 不需要设置的属性可以省略，大小和风格一定不能省略
  • 范例

二、文本属性

2.1. 简介

  • 文本的属性
    • 颜色

2.2. 颜色

  • 用color定义文本颜色
  • 开发中常用十六进制的属性值来设置颜色
  • 范例

2.3. 对齐

  • text-align属性用于设置元素内文本内容的水平对齐方式
  • 默认为左对齐left
  • 范例

2.4. 装饰

  • text-decoration属性可以给文本添加下划线underline、删除线overline、上划线line-through
  • none默认值，没有装饰，链接自带下划线，可以设置成none来去掉下划线
  • 范例

2.5. 缩进

  • text-indent属性用于指定文本第一行的缩进
  • em表示缩进1个文字大小的单位
  • px像素单位
  • 该属性值可以为负值
  • 范例

2.6. 行间距

  • line-height属性用于设置行间的距离
  • 行间距=上间距+文本高度+下间距
  • 范例

三、背景属性

3.1. 简介

  • 背景属性可以给页面元素添加背景样式
  • 背景属性可以设置
    • 背景颜色
    • 背景图片
    • 背景平铺
    • 背景图片位置
    • 背景图像固定等

3.2. 设置背景颜色

  • background-color
  • 默认颜色为透明transparent
  • 语法background-color: 颜色;
  • 范例

3.3. 背景颜色半透明

  • 可以使背景颜色变成半透明，其他内容不受影响
  • 取值范围为：0~1
  • 半透明是CSS3新增的属性
  • 语法：background: rgba(0, 0, 0, 0.5)
    • 前三个数值用于确定颜色
    • 最后一个数值用于确定透明度alpha
    • 可以将小数点前面的0省略掉，即.5
  • 范例

3.4. 设置背景图片

  • background-image
  • 常用于logo、装饰性的小图片、超大的背景图片、精灵图
  • 优点是便于控制位置
  • 语法background-image: url(图片位置)
  • 默认值为none，无背景图，background-image: none
  • 范例

3.5. 背景图片平铺

  • background-repeat
  • 对背景图片进行平铺
  • 值
    • repeat：横向和纵向都平铺（默认值）
    • no-repeat：不平铺
    • repeat-x：横向平铺
    • repeat-y：纵向平铺
  • 范例

3.6. 背景图片位置

  • background-position
  • 设置背景图片的位置
  • 值
    • 方位名词
    • 精确单位
    • 混合单位

3.6.1. 方位名词

  • 如果设置了两个参数，则没有前后顺序
  • 如果只设置了一个参数，则第二个值默认为水平居中
  • 竖直方向的值可以为：top、center、bottom
  • 水平方向的值可以为：left、center、right
  • 范例

3.6.2. 精确单位

  • 有前后顺序
  • 如果设置了两个数值，则第一个值为x轴坐标， 第二个值为y轴坐标
  • 如果只设置了一个数值，则该数值一定是x坐标，另一个数值默认为垂直居中
  • 范例

3.6.3. 混合单位

  • 有前后顺序
  • 可以将精确单位和方位名词混合使用
  • 第一个一定值是x轴坐标，第二个值一定是y轴坐标
  • 范例

3.7. 背景图像固定

  • background-attachment
  • 可以设置背景图像是否固定或随着页面滚动
  • 属性
    • scroll背景图像随对象内容滚动
    • fixed背景图像固定
  • 范例

3.8. 背景属性复合写法

  • 可以将关于背景的相关属性合并写在同一个background属性中
  • 简化了代码
  • 开发中常用
  • 简写时，没有特定的书写顺序
    • 一般顺序为：背景颜色 图片地址 图片平铺 图片滚动 图片位置
    • background: transparent url(1.jpg) repeat-y fixed top;
  • 范例
